## 阶段一：数学基础（第1-4周）

**目标**：掌握图形学所需的线性代数核心概念

### 每周任务

| 周 | 学习内容 | 练习项目 | 课程资源 |
|----|----------|----------|----------|
| 第1周 | 向量运算（加减、点积、叉积） | 用C++实现Vector3类 | 3Blue1Brown《线性代数的本质》P1-4 |
| 第2周 | 矩阵运算（乘法、转置、逆） | 实现4x4矩阵运算库 | 3Blue1Brown P5-10 |
| 第3周 | 线性变换（平移、旋转、缩放） | 实现相机变换矩阵 | Games101 L01-L03 |
| 第4周 | 空间变换（世界-视图-投影） | 实现简易3D管线 | 分析UE源码中FTransform |

### 阶段一必学内容

- [ ] 向量点积（Dot Product）- 用于计算夹角、投影
- [ ] 向量叉积（Cross Product）- 用于计算法线
- [ ] 矩阵乘法 - 变换的基础
- [ ] 齐次坐标 - 理解平移如何用矩阵表示
- [ ] LookAt相机矩阵 - 视图变换核心
- [ ] 投影矩阵 - 透视/正交变换
我的微调建议

  1. 向量基础（点积、叉积）
  2. 坐标系概念（先理解"空间"的概念）
  3. 矩阵变换（理解如何在空间间转换）
  4. MVP（理解渲染管线）
  5. 投影、反射（向量进阶）
  6. TBN（综合应用）
  7. 四元数（旋转专题）
  8. 相交检测（实用工具）

  原因：先理解"空间"的概念，再学矩阵会更容易理解

## 阶段二：Shader与渲染基础（第5-12周）

**目标**：掌握HLSL/Shader编程和渲染基础理论

### 每周任务

| 周 | 学习内容 | 练习项目 | 课程资源 |
|----|----------|----------|----------|
| 第5周 | HLSL基础语法、数据类型 | 写一个简单的颜色渐变Shader | Microsoft HLSL文档 |

## 关键技术点速查

### 线性代数（必须掌握）

| 概念 | 图形学用途 |
|------|------------|
| 向量点积 a·b | 计算夹角、投影、 specular强度 |
| 向量叉积 a×b | 计算法线、构建坐标系 |
| 矩阵乘法 | 组合多个变换 |
| 齐次坐标 | 实现平移变换 |
| LookAt矩阵 | 相机视图变换 |
| 投影矩阵 | 透视/正交变换 |

### 渲染基础（必须掌握）

| 概念 | 说明 |
|------|------|
| GBuffer | Deferred渲染的MRT数据 |
| BRDF | 双向反射分布函数 |
| Fresnel | 菲涅尔效应（掠射角反射强）|
| Mipmap | 纹理多级渐进 |
| Forward/Deferred | 两种渲染方式 |

## 关于DX12/Vulkan

**短期（第1-8个月）：不必要**

UE已经封装了RHI层，绝大多数TA工作不需要直接写DX12/Vulkan。

**长期（1年后）：可选**

- 想深入渲染管线、做渲染向专家 → 可以学习
- 建议先掌握Shader和UE渲染框架，再反向学习图形API

## 学习顺序总结

```
优先级高（必学）：
  线性代数 → Shader基础 → UE渲染框架 → C++

优先级中：
  Python工具开发 → 计算机图形学理论 → 性能优化

优先级低（后期）：
  DX12/Vulkan → 离线渲染 → 引擎架构
```
